# Particular

Unity project that explores the periodic table and atoms up close by using Unity particle system for protons, neutrons and electrons. You can play around with particle values to create ionic variants and isotopes.

The UI is entirely built from UIToolkit using various bindings, templates and custom elements.

Data is generated from local JSON files. I got this data years ago - can't remember where though. It was free and openly sourced, and has been edited since.

## How to use
- Load the BRView scene and run.
- Press the periodic table icon to pull up the table and select different elements
- Adjust particle values using the +/- buttons at the bottom left
- The '?' info button will display an information window with additional info on the selected element

Simple C# with standard `UnityEngine` library. Works with native Unity 6.2. Feel free to use.
